Point: The ID of the point in the point_controller

Wire Type: The length and type of wire for the control panel. Short = 10cm, Medium = 15cm, Long = 20cm, Double = 15cm with an extra LED

Node ID: Two digit binary for the i2c address of the point node for that point

Point ID: Four digit binary for the pin the individual point is wired to

Point Name: The name used within the code as a constant identifer for a point
